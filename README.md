# Sahibash

Sahibash is an original Afghanistan-focused classifieds marketplace.

## Run

```bash
npm install
npm run dev
```

## Work From E Drive

Project path: E:\sahibash

## Supabase env

Copy `.env.example` to `.env.local` and set keys.
